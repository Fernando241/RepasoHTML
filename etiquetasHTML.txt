TIPOS DE LISTAS

* listas ordenadas *
<ol>	
	<li>
	<li>
</ol>

* listas desordenadas *
<ul>
	<li>
	<li>
</ul>

<ul type="circle"> cambia a circulos
<ul type="square"> cambia a cuadrados
<ul type="none"> elimina los guiones de la lista

<ol type="A"> ordena por letras mayusculas
<ol type="a"> ordena por letras minusculas
<ol type="I"> ordena por numero romanos en mayuscula
<ol type="i"> ordena por numeros romanos en minuscula

tambien es importante recalcar que se pueden colocar listas dentro de listas

* listas de descripcion *
<dl>					lista de descripcion
	<dt>nombre</dt>			nombre del termino
		<dd>descripcion</dd>	descripcion del termino
</dl>

lo anterior es la forma de crear una lista de descripcion

TABLAS

<table border="1">	//Este atributo permite establecer el
			gruesor del borde de la tabla

	<caption>Nombre de la Tabla</caption>

	<thead>	//Esta etiqueta no cambia el aspecto visual de la
		tabla pero se utiliza para colocar el encabezado y
		resaltarlo algo MUY UTIL PARA DAR ENFASIS
		EN LOS BUSCADORES DE BUSQUEDA Y SEO
	<tr>			
	      	<th colspan="7">//Este atributo permite establecer
				la cantidad de columnas que va ocupar
				esta Titulo
		</th>
	      	<th rowspan="3">//Este atributo permite establecer
				la cantidad de Filas que va ocupar
				esta Titulo
		</th>
	</tr>

	<tr>	//---tableRow -fila de tabla---
		<th> Titulos </th>	
		<td> Datos </td>

				//la cantidad de th,tr dependen 
				de de la cantidad de columnas de
				nuestra tabla

	</tr> 	//Cierre de la fila //la cantidad de tr depende
				de la cantidad de filas de 
				nuestra tabla
</table>	//Cierre de la tabla

FORMULARIOS EN HTML

<form	action = "scrip.php" method = "" define la privacidad de los datos a enviar
					"GET" sin importancia relevante
					"POST" información de relevancia y cuidado
						>

	<input type =		//permite identificar el tipo de dato a ingresar

			"text"		//ingresa datos de texto
			"password" 	//ingresa datos ocultos
			"email" 	//ingresa correos electronicos
			"date" 		//permite enviar fechas
			"radio" 	//permite seleccionar una entre varias opciones
			"checkout"	//permite selecionar varias opciones al tiempo
			"submit" 	//se oprime para enviar datos
			"reset"		//permite limpiar todos los campos llenos

		id = " "	//sirve para identificar

		name = " "	//en text,password,email,date es lo que se envia al servidor
	
		value = " "	//en otros atributos es lo que se envia al servidor

		placeholder = " "	//leyenda dentro de los cuadros para orientar al usurio
	
		requiered 	//no permite enviar el formulario si este campo esta vacio
		
		autofocus	//permite colocar el cursor y resaltar este campo
		
	>

	<fieldset>
		enmarca todo lo que esta dentro en el formulario
		<legend> 
			permite escribir un título que identifica lo que esta encerrado en el fieldset
		</legend>
	</fieldset>

	<textarea	//cuadro de texto grande para descripciones

			cols = "ancho en columnas"
			rows = "alto en filas"
			placeholder = "Descripción orientando al usuario"
		>
			texto orientativo borrable
	</textarea>

	<select name="" id = "">	//crea una lista de seleccion
		<option value = "valor a enviar">	opcion 1	</option>
		<option value = "valor a enviar">	opcion 2	</option>
		<option value = "valor a enviar">	opcion 3	</option>
	</select>

</form>

AUDIO

---forma extendida--------------------------
<audio	//permite insertar audio en la pagina
	controls	//permite visualizar los archivos de audio
	autoplay	//hace que el audio se reproduzca automaticamente
	muted		//silencia el audio que se reproduce automaticamente
	loop		//hace que el audio se repita automaticamente
		>

	<source
		src = "direccion del audio a insertar"
		type = "audio/mp3"	//depende del formato a usar
			>
</audio>

	Nota: esta forma extendida me  permite colocar varios audio con la etiqueta source con diferentes formatos
---forma directa

<audio src"direccion del audio a insertar" -otros atributos- ></audio>
